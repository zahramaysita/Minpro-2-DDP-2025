# **1. Penjelasan Flowchart**
**Alur Program**
1. mulai program
2. pilih role
3. login
4. terdapat 6 pilihan menu yaitu :
    1. Lihat Produk
    2. Tambah Produk
    3. Update Produk
    4. Hapus Produk
    5. Total Pengeluaran
    6. keluar
